## 1. 目的とスコープ

### 1.1 目的

- 全属性（剣・盾を含む **10 属性**）の **戦闘中の行動・効果・数値** を単一の参照源として定義する。
- [プロトタイプ開発指示書 §8](./PROTOTYPE_DEVELOPMENT_SPEC.md#8-戦闘ルール) の骨格（5枚・前衛2/後衛3・1ターン）に載る **属性レイヤ** を記述する。

### 1.2 プロトタイプ現行との関係

| 区分 | 内容 |
|------|------|
| **実装済み** | 剣・盾・力・弓（`attack` / `defense` / `power` / `bow`） |
| **仕様確定・未実装** | 両・毒・癒・氷・嵐・忍 |
| **属性決定（作成時）** | 現行: hash + 色比率（剣/盾のみ）。将来: AI 漢字1文字（[PROTOTYPE §5.4](./PROTOTYPE_DEVELOPMENT_SPEC.md#54-属性決定)） |
| **表示** | 漢字1文字 + `ATTRIBUTE_META`（`src/config/attributes.ts`） |

### 1.3 前衛・後衛の基本

- **前衛**: 原則 **近接攻撃** が可能（属性ごとに追加行動あり。§4 参照）。
- **後衛**: 属性ごとに行動可否が異なる（§4 参照）。
- **将来方針**: 前衛でも「近接 OR 特殊」の **1ターン1行動** に統一する案あり（現仕様は **属性ごと** に前衛/後衛の行動を定義）。

## 2. 共通ルール

### 2.1 1ターン1行動（行動カード）

| 項目 | 内容 |
|------|------|
| 原則 | 各陣営 **1ターンに1枚だけ** 行動カードを選ぶ |
| 行動の種類 | 近接攻撃・弓攻撃・両・嵐・癒・盾付与 など（属性による） |
| 排他 | 嵐・弓・両・近接は **そのターンの行動カード1枚の攻撃行動**。他カードは同ターン行動しない |
| 付与系 | 毒・氷は **近接の行動カード** で戦った相手に、攻撃と同時に付与 |
| 忍（後衛） | ステルス中は **行動カードに選べない**（前衛に上がってから行動） |

### 2.2 ダメージの基準

| 種別 | 基準 |
|------|------|
| **原則** | バトル中の攻撃・付与ダメージ（近接、弓、両副攻撃、嵐など）は **付与者の currentBp** |
| **例外（毒 DoT）** | 各毒スタックは **付与時点の付与元 maxBp × 20%**（§5.1） |
| **回復上限** | 癒による回復は **maxBp まで**（オーバーヒールなし） |
| **表示** | 対人戦で相手が読みやすいよう、攻撃は原則 **今カードに表示されている BP** を基準とする |

### 2.3 近接攻撃と反撃

| 状況 | 攻撃側 | 反撃側 |
|------|--------|--------|
| **相打ち**（同じ2枚が互いに近接の行動カード） | currentBp **100%** | currentBp **100%** |
| **一方的**（片方だけが近接の行動カード、または別の相手を攻撃中の前衛が横から殴られる） | currentBp **100%** | currentBp **50%** |

- 相打ち判定: 同一ターンに互いが相手を近接ターゲットに選んだ **同じ2枚** の組（実装: `bidirectional` フラグ）。
- 弓・両副攻撃・嵐は **反撃ルールの対象外**（無反撃または別規定）。

### 2.4 盾（全属性共通）

| 項目 | 内容 |
|------|------|
| 効果 | **1回限り**、ダメージを 0 にして消費 |
| 嵐 | 盾があれば **盾消費・ダメージ 0** |
| 攻撃側の盾 | 近接で攻撃した側が盾を持つ場合、反撃ダメージ 0・攻撃側盾消費（剣/盾 Lv0 ルール維持） |

## 3. 属性一覧・解放

### 3.1 一覧

| 内部 ID | 漢字 | 解放 Lv | tie-break 優先度 | 実装 |
|---------|------|---------|------------------|------|
| `attack` | 剣 | 1 | 1 | 済 |
| `defense` | 盾 | 1 | 2 | 済 |
| `power` | 力 | 6 | 3 | 済 |
| `bow` | 弓 | 11 | 4 | 済 |
| `dual` | 両 | 16 | 5 | 未 |
| `poison` | 毒 | 21 | 6 | 未 |
| `heal` | 癒 | 26 | 7 | 未 |
| `ice` | 氷 | 31 | 8 | 未 |
| `storm` | 嵐 | 36 | 9 | 未 |
| `ninja` | 忍 | 41 | 10 | 未 |

- tie-break: 数値が **大きいほど先** に解決（§7）。
- Lv46 以降: 将来属性用（[PROTOTYPE §5.9](./PROTOTYPE_DEVELOPMENT_SPEC.md#59-ユーザーレベル解放5nm-統一)）。

### 3.2 作成時の属性

- **新規作成時のみ** 属性決定。編集では変更不可（[PROTOTYPE §5.8](./PROTOTYPE_DEVELOPMENT_SPEC.md#58-新規作成と編集)）。
- 候補属性は **作成時点で解放済み** のもののみ（将来 AI 属性時も同様）。

## 4. 各属性の詳細

### 4.1 剣（attack）— 実装済み

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃**（行動カード）。敵前衛の左右どちらか |
| 後衛 | 行動不可 |

- 特殊能力なし。§2.3 の近接・反撃ルールに従う。

### 4.2 盾（defense）— 実装済み

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃** または **盾付与** のどちらか1つ（行動カード） |
| 後衛 | **盾付与** のみ（付与可能な対象がいる場合） |

| 項目 | 内容 |
|------|------|
| 初期盾 | バトル開始時から **盾1枚**（前衛・後衛問わず） |
| 盾付与 | 盾なしの **味方1体**（**自分不可**）。**1戦闘1回**（防御カードごと） |
| 操作（前衛） | 行動カード選択後、敵前衛タップ＝攻撃、盾なし味方タップ＝盾付与 |

### 4.3 力（power）

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃**のみ |
| 後衛 | 行動不可 |

| 項目 | 内容 |
|------|------|
| 特殊能力 | **なし**（高 BP で戦う） |
| BP 算出 | 他属性と同様 hash＋密度でレンジ内を決定したうえで、**中心 BP に `POWER_BP_RATIO`（1.5）** を乗じたレンジを使用（[PROTOTYPE §5.5](./PROTOTYPE_DEVELOPMENT_SPEC.md#55-bp-決定)）。レア倍率は他属性と同タイミング |

### 4.4 弓（bow）— 実装済み

| 項目 | 内容 |
|------|------|
| BP 算出 | 剣比 **`BOW_BP_RATIO`（0.6）**。レア倍率は他属性と同タイミング |
| 行動 | **弓攻撃のみ**（前衛・後衛問わず） |
| 弓攻撃 | 生存している **敵1体** を選択。**反撃なし**。与ダメ = currentBp **100%** |
| 対象 | 敵前衛・後衛のいずれも可。**忍のステルス中は狙えない** |
| 近接 | **不可** |

### 4.5 両（dual）

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃**（行動カード）＋副攻撃 |
| 後衛 | 行動不可 |

| 項目 | 内容 |
|------|------|
| 主対象 | 通常の近接（§2.3）。毒・氷の付与も **主対象のみ** |
| 副対象 | **もう片方の敵前衛**（主対象と **同時**） |
| 副ダメージ | 自分の currentBp × **50%**、**無反撃** |
| 副対象の盾 | あれば **盾消費**（ダメージ 0） |
| 忍 | ステルス中は **近接の主対象にも副対象にもならない** |

### 4.6 毒（poison）

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃** ＋ 毒付与 |
| 後衛 | 行動不可 |

| 項目 | 内容 |
|------|------|
| 付与 | 近接で **戦った** 相手に毒（主対象のみ。両の副対象には付与しない） |
| スタック | **付与元カード（cardId）ごと** に独立して重ねがけ可 |
| DoT | **毎ターン開始時**、各スタック = **付与時点の付与元 maxBp × 20%** を合算 |
| 持続 | 対象が倒れるまで。 **付与者が撃破されても** 既存スタックは継続 |
| 近接本体 | currentBp 基準（§2.2） |

### 4.7 癒（heal）

| 位置 | 行動 |
|------|------|
| 後衛 | **回復**（行動カード） |
| 前衛 | **近接攻撃** または **回復** のどちらか1つ |

| 項目 | 内容 |
|------|------|
| 回復量 | 回復時点の **自分の currentBp** |
| 対象 | **既にダメージを受けた味方のみ**（満タン不可）。**自分も可** |
| 上限 | **maxBp まで**（オーバーヒールなし） |
| 回数 | **1枚あたり1戦闘2回まで** |
| タイミング | 行動カードで癒を選んだターン、**盾・攻撃より先** に解決（§6） |
| 凍結中の味方 | **回復可能**（回復を受けると忍のステルス解除対象になりうる — §5.3） |

### 4.8 氷（ice）

| 位置 | 行動 |
|------|------|
| 前衛 | **近接攻撃** ＋ 凍結付与 |
| 後衛 | 行動不可 |

| 項目 | 内容 |
|------|------|
| 付与 | 近接で **戦った** 相手に凍結（主対象のみ） |
| 効果 | **次ターン開始〜次ターン終了** まで行動不能。 **次々ターン** から行動可能 |
| 再凍結 | 凍結中を再度近接 → **タイマーリセット**（再び「次ターン丸ごと凍結」） |
| 攻撃対象 | 凍結中も **攻撃・回復の対象に選べる** |
| 毒 DoT | 凍結中も **毎ターン開始時に毒ダメージは入る** |

### 4.9 嵐（storm）

| 位置 | 行動 |
|------|------|
| 後衛 | **嵐**（行動カード、**1戦闘1回**） |
| 前衛 | **近接攻撃** または **嵐** のどちらか1つ |

| 項目 | 内容 |
|------|------|
| ダメージ | 自分の currentBp × **25%** を、生存敵から **ランダム2体** に（**必ず別ユニット**。1体しかいなければ1体） |
| 盾 | 盾があれば **盾消費・ダメージ 0** |
| 忍 | ステルス中も **当たりうる**（ダメージ＋ステルス解除） |

### 4.10 忍（ninja）

| 位置 | 行動 |
|------|------|
| 後衛 | **ステルス維持**（行動カードに **選べない**） |
| 前衛 | **近接攻撃**のみ |

| 項目 | 内容 |
|------|------|
| ステルス中 | 敵の **攻撃対象にならない**（弓・両副対象・嵐のランダム対象からも除外） |
| 初回無反撃 | **1戦闘1回限り**。自分が **初めて敵を攻撃した** 近接は無反撃 |
| ステルス解除 | ① **自ら敵を攻撃した瞬間**（その1回は無反撃） ② **外部から何らかの効果・ダメージを受けた瞬間**（近接・弓・嵐・毒 DoT・癒・盾付与・凍結など） |
| 再ステルス | **なし** |

## 5. 状態異常・バフ

### 5.1 毒（スタック）

- `BattleUnit` 上で **付与元 cardId ごとのスタックリスト** を保持する想定。
- ターン開始フェーズ（§6 ①）で合算 DoT を適用。

### 5.2 凍結

- 行動カード選択不可のターンを `frozenUntilTurn` 等で管理。
- 攻撃対象としては選択可能（§4.8）。

### 5.3 ステルス（忍）

- `stealthActive` 等。解除条件は §4.10。
- **癒・盾・嵐・ダメージ全般** で解除。

### 5.4 盾

- 盾属性の初期盾・付与に加え、全属性が `hasShield` を持ちうる。

## 6. ターン処理順

```
① ターン開始：毒 DoT（全ユニット）

② 行動カード選択（各陣営1枚）
   ※ 凍結中のユニットは選択不可

③ 癒（行動カードで癒を選んだ側）
   ※ currentBp 高い順。同率は §7

④ 盾付与（行動カードで盾付与を選んだ側）
   ※ currentBp 高い順。同率は §7

⑤ 攻撃フェーズ
   嵐・弓・近接・両 等（行動カードの currentBp 高い順。同率は §7）
   ※ 近接は §2.3。複数近接は bidirectional 判定後、1戦闘ずつ

⑥ 撃破・墓地移動

⑦ 前衛補充
   ※ 弓が前衛に上がっても BP は変化しない
```

## 7. 同率時の解決順（tie-break）

癒・盾付与・攻撃フェーズなど **currentBp 順** を使う箇所で、BP が同率のとき:

| 順位 | 条件 |
|------|------|
| 1 | **currentBp**（高い方が先） |
| 2 | **属性優先度**（§3.1 の表。大きい方が先） |
| 3 | **レア度**（高い方が先: SR > R > N …） |
| 4 | **限界突破 ★**（多い方が先） |
| 5 | **ランダム**（50%。対人戦の公平性のため） |

|  | 忍（ステルス） | 盾 | 凍結 | 毒 DoT |
